# Superstore Sales Data Analysis

## Project Overview
This project analyzes a retail dataset to extract business insights using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI

## Key Analysis
- Total Sales and Profit Analysis
- Sales by Region and Category
- Monthly Sales Trend
- Top Selling Products
- Profit Analysis
- Shipping Time Analysis
- Discount Impact on Profit

## Power BI Dashboard
The dashboard includes:
- KPI Cards (Sales, Profit, Orders)
- Sales by Region and Category
- Profit by Category
- Monthly Sales Trend
- Top Products
- Interactive Filters

## Insights
- Technology category generates highest sales and profit
- Some regions perform better than others
- High discounts reduce profit
- Certain products contribute more revenue
